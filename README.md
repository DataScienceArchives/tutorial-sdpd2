# tutorial-sdpd2

**URL**: <https://datasciencearchives.github.io/tutorial-sdpd2/>.

Esta es una colección de tutoriales prácticos sobre diferentes herramientas
software, entornos de programación y gestión de proyectos utilizados en la
asignatura *Sistemas Distribuidos de Procesamiento de Datos II*, impartida en
3º curso del Grado en Ciencia e Ingeniería de Datos de la URJC.
